FROM THE PROJECT PAGE 
--------------------- 
This module provides the ability to
shroud various elements of the node edit form from users who don't have the
bypass permission.

MENU SUPPORT 
------------ 
form_shroud provides Drupal core menu support,
allowing users to shroud links on a menu's overview page.

ORGANIC GROUPS MENU SUPPORT 
--------------------------- 
og_menu - http://drupal.org/project/og_menu

form_shroud provides og_menu support for your Organic Groups menu overview
page(s).
